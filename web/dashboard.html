03Performance du modèle déployé

Cadre éthique. PhishLab est un outil défensif. Tous les exemples sont synthétiques, dans un bac à sable : aucun envoi réel, aucune cible réelle, aucune collecte de données. Le score affiché provient d'un modèle linéaire embarqué (approximation fidèle du détecteur entraîné) exécuté entièrement dans le navigateur ; pour des scores exacts en production, l'API Flask sert le même modèle calibré.
